Our Story

Welcome to MiniBlog, a platform dedicated to sharing knowledge and insights about web development, design, and technology. This project was created as part of a web development assignment to demonstrate skills in HTML, CSS, and JavaScript.

MiniBlog aims to showcase responsive design principles, semantic HTML markup, modern CSS techniques, and JavaScript interactivity in a real-world web application.

Features

Responsive Design

Our blog is designed to work seamlessly across all devices, from mobile phones to desktop computers.

Clean Code

Built with semantic HTML5, modern CSS3, and vanilla JavaScript for optimal performance.

Search Functionality

Easily find the content you're looking for with our search feature.

Mobile First

Designed with a mobile-first approach to ensure great experience on all screen sizes.
